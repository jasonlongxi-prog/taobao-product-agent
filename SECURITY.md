# 安全说明

## 报告问题

如果你发现扩展读取了预期范围之外的数据、下载了非商品资源，或脚本存在命令执行、路径穿越、敏感信息泄露等问题，请通过 GitHub Security Advisory 私下报告，不要在公开 Issue 中附上 Cookie、订单截图或个人资料。

## 安全边界

- 项目不需要淘宝 Cookie、账号密码或 API 密钥；任何索取这些信息的第三方版本都不属于本项目的标准流程。
- 不要通过修改代码绕过登录、验证码、访问验证或平台反爬限制。
- 扩展请从源码自行加载，安装前检查 `manifest.json` 权限。
- OCR 输入可能来自不可信网页；请在普通用户权限下运行，并保持 Pillow、Requests、Tesseract 等依赖更新。

## 支持范围

本项目用于本地素材整理与分析，不保证淘宝/天猫页面结构长期不变，也不保证采集分类百分之百准确。

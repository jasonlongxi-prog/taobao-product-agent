# Chrome 采集助手使用说明

## 安装

1. 下载或克隆本仓库。
2. 在 Chrome 地址栏打开 `chrome://extensions/`。
3. 开启右上角“开发者模式”。
4. 点击“加载已解压的扩展程序”。
5. 选择仓库中的 `assets/chrome-extension/` 目录。
6. 将“淘宝商品分析助手”固定到工具栏。

## 采集商品

1. 在 Chrome 中正常登录淘宝或天猫。本项目不需要也不应获取你的密码或 Cookie。
2. 打开一个商品详情页，确认主图、规格和详情能够正常显示。
3. 点击扩展图标，再点击“采集当前商品”。
4. 页面会自动滚动以触发懒加载；完成前不要关闭标签页。
5. 文件会保存到 Chrome 当前下载目录下的 `TaobaoAgent/<商品ID>/`。

若找不到下载结果，请打开 `chrome://settings/downloads` 查看 Chrome 当前下载位置。

## 结果结构

```text
TaobaoAgent/<商品ID>/
├── manifest.json
├── main/
├── sku/
└── detail/
```

图片分类由页面位置、元素上下文和尺寸启发式判断，使用前请结合清单与原页面复核。

## 隐私与版权

扩展只在用户主动点击时处理当前商品页，并不会导出账号密码、Cookie、订单、地址或聊天记录。商品素材可能受版权、商标或肖像权保护，请仅在获得授权或法律允许的范围内使用。
